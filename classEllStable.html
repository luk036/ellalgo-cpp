Detailed Description

template<typename Arr>
class EllStable< Arr >

Ellipsoid Search Space.

\[ \mathcal{E} = \{x \mid (x - x_c)^T Q^{-1} (x - x_c) \le \kappa\} \]

Keep $Q$ symmetric but no promise of positive definite
