Detailed Description

Oracle for FIR lowpass filter design.

This example is taken from Almir Mutapcic in 2006: 

min   γ
s.t.  L²(ω) ≤ R(ω) ≤ U²(ω), ∀ ω ∈ [0, π]
      R(ω) > 0, ∀ ω ∈ [0, π]
