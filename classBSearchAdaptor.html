Detailed Description

template<typename Oracle, typename Space>
class BSearchAdaptor< Oracle, Space >

Binary search adaptor wrapping a cutting-plane feasibility oracle.

Uses cutting_plane_feas() as a sub-routine to test feasibility at each candidate value, enabling binary search on a parameter.
